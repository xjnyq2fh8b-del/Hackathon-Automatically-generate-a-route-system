# Static V2 Backup

This folder stores a copy of the static web version before the first-stage mobile information architecture refactor.

## Contents

- `index.html`: previous static page entry.
- `package.json`: previous local script config.
- `server.mjs`: previous local static server.
- `README.md`: previous frontend workspace notes.
- `src/app.js`: previous UI/data logic.
- `src/styles.css`: previous visual system.

## Restore Rule

To restore this backup, copy these files back to `frontend/` while preserving the same relative paths.
